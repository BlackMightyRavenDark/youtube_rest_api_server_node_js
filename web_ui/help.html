GET

/api/get_video_info

Получить информацию о видео. Будут использованы куки по-умолчанию, если они установлены на сервере.

Query:
Параметр Тип Значение Обязательный? Описание
video_id string ID видео на YouTube Да ID запрашиваемого видео на YouTube. Обычно состоит из 11 строковых символов.
api_client_name string Имя внутреннего клиента YouTube Нет

ID внутреннего клиента YouTube, через который будет получена информацию о видео и ссылки для скачивания. Разные клиенты могут выдавать разные форматы для скачивания.

Значение по-умолчанию: auto
requested_data string Одно или несколько значений через запятую Нет

Одно или несколько значений, перечисленных через запятую, указывающие, какие поля должны содержаться в ответе на запрос. Если по какой-то причине запрашиваемое поле получить невозможно, оно не будет включено в ответ. Если будет передано хотя-бы одно неподдерживаемое значение, сервер вернёт ошибку с кодом 400. Если будет передана пустая строка или последовательность из запятых (с пробелами или без), будет использовано значение по-умолчанию. Если в списке передано значение "urls", будет предпринята попытка получить и расшифовать ссылки для скачивания, если это необходимо. Если значение не передано или попытка не увенчается успехом, ссылки для скачивания видео могут оказаться недействительными или вовсе отсутствовать. Если в списке передано значение "all", в ответе будет содержаться вся возможная информация о видео, независимо от того, какие ещё значения переданы.

Возможные значения: raw_video_info, parsed_video_info, urls, web_page, all

Значение по-умолчанию: all
